# VIGILANTEye Logo

Place the VIGILANTEye logo image here.

## File Name
- `vigilanteye-logo.png` or `vigilanteye-logo.jpg`

## Recommended Specifications
- **Format**: PNG (with transparency) or JPG
- **Size**: 400x200px or larger (maintain aspect ratio)
- **Background**: Transparent PNG preferred, or white background
- **File Size**: Under 500KB

## Usage
The logo will be used in:
- Navigation bar (small version)
- Hero section (large version)
- Footer (medium version)

## File Location
Save your logo as: `public/images/logo/vigilanteye-logo.png`
